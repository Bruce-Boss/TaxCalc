# TaxCalc - 个税现金流计算器

TaxCalc 个税现金流计算器 的 微信小程序。

### 背景介绍

易用的个人现金流计算器，帮助你了解每个月收入，管理全年现金流。

2019年颁布了新税法，改成 “累计预扣预缴”，造成的结果是每个月实际到手收入会持续下降。本个税现金流计算器可以计算全年现金流情况。
